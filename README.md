# Lull

A lightweight ambient sound player for macOS. Mix rain, wind, fire, ocean, birds, and café noise freely — find your perfect focus environment.

🌐 **[lull-app website](https://olddsun.github.io/lull-app/)**

---

## Download

Go to the [Releases page](https://github.com/Olddsun/lull-app/releases/latest) and download:

- **Apple Silicon (M1/M2/M3):** `Lull-x.x.x-arm64.dmg`
- **Intel Mac:** `Lull-x.x.x.dmg`

## First time opening

Since Lull is not signed with an Apple Developer certificate, macOS may show a security warning on first launch.

1. Open **System Settings**
2. Go to **Privacy & Security**
3. Scroll down and click **Open Anyway**

This is a one-time step. Lull contains no malware.

---

## Features

- 🎚️ Mix 6 sound sources: Rain, Wind, Birds, Fire, Waves, Café
- 🌲 Quick scene presets: Forest, Rainy Night, Café, Beach
- ✨ Auto mode — gently shifts the mix over time
- 📌 Always on Top — floats above your workspace
- 🔇 Hide window and keep playing in the background
- 🖥️ Menu bar control

## Requirements

- macOS 12 or later

## Run from source

```bash
git clone https://github.com/Olddsun/lull-app.git
cd lull-app
npm install
npm start
```
