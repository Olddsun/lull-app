# Lull

A lightweight ambient sound player for macOS. Mix rain, wind, fire, ocean, birds, and café noise freely — find your perfect focus environment.

一款輕量的 macOS 環境音播放器。自由混合雨聲、風聲、火焰、海浪、鳥鳴、咖啡廳白噪音，打造屬於你的專注空間。

🌐 **[Official Website / 官方網站](https://olddsun.github.io/lull-app/)**

## First time opening / 第一次開啟

Since Lull is not signed with an Apple Developer certificate, macOS may show a security warning on first launch.

由於 Lull 未經 Apple Developer 憑證簽署，macOS 第一次開啟時可能會顯示安全性警告。

1. Open **System Settings** / 開啟**系統設定**
2. Go to **Privacy & Security** / 前往**隱私權與安全性**
3. Scroll down and click **Open Anyway** / 往下滑，點擊**仍要開啟**

This is a one-time step. Lull contains no malware.

此步驟只需操作一次，Lull 不含任何惡意程式。

## Features / 功能特色

- 🎚️ Mix 6 sound sources: Rain, Wind, Birds, Fire, Waves, Café / 6 種環境音混音
- 🌲 Quick scene presets: Forest, Rainy Night, Café, Beach / 快速場景切換
- ✨ Auto mode — gently shifts the mix over time / 自動調整模式
- 📌 Always on Top — floats above your workspace / 懸浮視窗
- 🔇 Hide window and keep playing in the background / 隱藏視窗繼續播放
- 🖥️ Menu bar control / 選單列控制

## Requirements / 系統需求

- macOS 12 or later / macOS 12 以上

## Run from source / 從原始碼執行

```bash
git clone https://github.com/Olddsun/lull-app.git
cd lull-app
npm install
npm start
```
